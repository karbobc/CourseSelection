# Course Selection System
Saitama's curriculum design.



## Features

- database: `SQL Server`
- UI design: `PyQt5`



## Quick Start

1. clone this repo.

```shell
git clone https://github.com/karbob666/CourseSelection.git
cd CourseSelection
```

2. using `pip` to install dependencies.

```shell
pip install -r requirements.txt
```

3. run `main.py`

```shell
python main.py
```
